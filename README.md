# Verilog DPI

## Building
```
./compile.sh
```

## Usage
```
Usage: ./obj_dir/test_bin [OPTION]

Options:
    +arg0=NUM           set first argument
    +arg1=NUM           set second argument
    +file=FILE          read "arg0 arg1" from FILE
    +random_mode=NUM    generate NUM pseudorandom pairs (arg0, arg1) as input
```
